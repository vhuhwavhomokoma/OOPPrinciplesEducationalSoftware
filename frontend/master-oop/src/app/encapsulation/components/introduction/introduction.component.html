Introduction to Encapsulation Understanding the Basics
  • Definition:

    Encapsulation means keeping some information hidden (private) and only allowing access to it through specific ways (methods). It's like a TV remote — you press buttons (methods), but you don’t see or control the complex circuits inside (data).

  • OOP:

    Object-Oriented Programming is a way to organise code by creating objects that represent real-world things, like a person, a car, or a book. Each object has attributes (data) and actions (methods) it can perform.
