Inheritance

Inheritance is one of the core principles of Object-Oriented Programming (OOP). Inheritance is a mechanism that allows one class (called a child class or subclass) to inherit properties and behaviors (i.e., attributes and methods) from another class (called a parent class or superclass).

The class whose properties and behaviours are inherited is referred to as the Parent class/Super class.

The class that inherits the properties and behaviours is referred to as the Child class/subclass.


  • The child class (or subclass) inherits attributes and methods from the parent class (or superclass). This means that the child class can use all the functionality of the parent class, while also extending or modifying it to meet its specific needs.
  • Method Overriding can be used to redefine a method in the child class that is inherited from the parent class, allowing for behaviour that meets the child class' specific needs
  • Inheritance encourages reusability, meaning you don't need to rewrite code for common features.
