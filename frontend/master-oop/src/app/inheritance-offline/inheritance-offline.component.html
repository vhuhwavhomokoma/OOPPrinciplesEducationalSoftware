Inheritance Offline Activity

Preconditions:

  • This activity should be done with a minimum of 9 participants.
  • The participants should be in person.
  • The activity should be done in a classroom or outside.
  • A pen and paper are required.

Activity Description:

  • Students are divided into groups.
  • Â Each group is assigned a single Animal from the group of Animal species: Elephant, Octopus, Lion, Eagle, Shark, Scorpion, Rhino, Rabbit, Duck, Sheep.
  • Each group comes up with attributes and behaviours for their assigned animal.
  • Each group then presents the attributes for their assigned animal and the attributes that they think are common with other groups.
  • All groups then consolidate the common attributes and assign them to the Animal super class.

Summary

  • Child Class/SubClass: A class that inherits attributes and methods from the parent class.
  • Parent Class/SuperClass: A class whose attributes and methods are inherited by the child class.
  • There are five types of inheritance: Single Inheritance, Multiple Inheritance, Multilevel Inheritance, Hierarchical Inheritance, Hybrid Inheritance.
  • Method Overloading: allows methods in the child class to perform similar but slightly different tasks from the parent class.
  • Reusability: is the practice of designing and writing code in a way that allows it to be used again in different contexts without modification.
